Connect your free database

This takes about 5 minutes. Everything below is free — no credit card needed. Follow these steps exactly:

1
Go to supabase.com, click Start your project, and sign up for free.
2
Click New project. Name it the-rail. Choose any region. Set a database password (save it somewhere). Click Create new project and wait ~1 minute.
3
In the left sidebar, click SQL Editor. Paste the code below and click Run.
create table closet_items ( id bigserial primary key, created_at timestamptz default now(), name text, category text, color text, image_data text, confidence float ); alter table closet_items enable row level security; create policy "Open access" on closet_items for all using (true) with check (true);
4
In the left sidebar, click Project Settings → API. Copy the Project URL and paste it below.
5
On the same page, copy the anon / public key (under "Project API keys") and paste it below.
