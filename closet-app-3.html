Connect your free database

Takes about 5 minutes, free, no credit card. Follow these steps exactly:

1
Go to supabase.com → Start your project → sign up free.
2
Click New project, name it the-rail, set a password, create it, wait ~1 minute.
3
Left sidebar → SQL Editor → paste this → click Run.
-- Profiles table (who's a client vs. the stylist) create table profiles ( id uuid references auth.users primary key, email text, full_name text, role text check (role in ('client','stylist')), created_at timestamptz default now() ); alter table profiles enable row level security; create policy "Authenticated can view profiles" on profiles for select using (auth.role() = 'authenticated'); create policy "Users can insert own profile" on profiles for insert with check (auth.uid() = id); -- Closet items, scoped per client create table closet_items ( id bigserial primary key, created_at timestamptz default now(), client_id uuid references auth.users not null, name text, category text, color text, image_data text, confidence float ); alter table closet_items enable row level security; create policy "Clients manage own items" on closet_items for all using (client_id = auth.uid()) with check (client_id = auth.uid()); create policy "Stylists view all items" on closet_items for select using ( exists (select 1 from profiles p where p.id = auth.uid() and p.role = 'stylist') );
4
Left sidebar → gear icon Settings → API (or "Data API"). Copy the Project URL.
5
Same page — copy the anon / public key.
