Connect your free database

Takes about 5 minutes, free, no credit card. Follow these steps exactly:

1
Go to supabase.com → Start your project → sign up free.
2
Click New project, name it the-rail, set a password, create it, wait ~1 minute.
3
Left sidebar → SQL Editor → paste this → click Run. If tables already exist from before, this clears them first.
drop table if exists closet_items cascade; drop table if exists profiles cascade; create table closet_items ( id bigserial primary key, created_at timestamptz default now(), client_name text not null, name text, category text, color text, image_data text, confidence float ); alter table closet_items enable row level security; create policy "Open access" on closet_items for all using (true) with check (true);
4
Left sidebar → gear icon Settings → API (or "Data API"). Copy the Project URL.
5
Same page — copy the anon / public key.
